1. **Download the Repository**  
   Click the green **"Code"** button on the repository page, then select **"Download ZIP"**.

2. **Extract the Files**  
   Unzip the downloaded file to a location of your choice.

3. **Locate the Relevant Fix**  
   Inside the repository, you'll find fixes tailored to specific maps:
   - `Whitewidow`
   - `AK47 Weed Shop`

4. **Install the Fix**  
   Navigate to the stream files of the MLO's Petshop & White Widow (this applies only to the white widow MLO, not the ak47 weed shop), replace the files from this repository in **both** resources stream folders.
